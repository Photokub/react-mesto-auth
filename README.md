# Проект: Место - REACT

### Обзор

Адаптивная персонализированная галерея с интерактивными элементами.

Брейкпоинты адаптивов находятся на отметках:
* 984px
* 620px
* 586px

### Инструменты
* Язык: Java Script.
* Библиотека: React. 
* Сборка с помощью Webpack.
